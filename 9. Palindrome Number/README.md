# 🪞 LeetCode Problem 9 - Palindrome Number

## 📌 Problem Description

Given an integer `x`, return `true` if `x` is a palindrome, and `false` otherwise.

A palindrome is a number that reads the same forward and backward.

### Example

```text
Input: x = 121
Output: true

Input: x = -121
Output: false

Input: x = 10
Output: false
```

---

## 💡 My Approach

My solution uses a simple and readable approach:

1. Convert the integer into a string.
2. Reverse the string using Python slicing (`[::-1]`).
3. Compare the reversed string with the original string.
4. If they are equal, return `True`; otherwise, return `False`.
